# 📊 Intelligent Sales Analytics Dashboard

An end-to-end Sales Analytics Web Application designed to help businesses explore datasets, analyze sales performance, identify customer trends, and make data-driven decisions through interactive dashboards and visualizations.

---

## 🚀 Project Overview

The Intelligent Sales Analytics Dashboard enables users to upload sales datasets and instantly generate valuable business insights. The application provides interactive dashboards, customer analytics, sales performance monitoring, and data exploration tools in a secure authenticated environment.

This project combines data analytics and web development concepts to create a business intelligence solution suitable for sales reporting and decision-making.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/intelligent-sales-analytics-dashboard.git
```

### Navigate to Project

```bash
cd intelligent-sales-analytics-dashboard
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Server

```bash
python manage.py runserver
```

### Open Browser

```text
http://127.0.0.1:8000/
```
